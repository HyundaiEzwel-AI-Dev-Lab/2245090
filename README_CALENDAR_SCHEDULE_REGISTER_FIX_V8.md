# PMS Calendar Schedule Register Fix v8

캘린더/일정 등록 관련 화면 문제를 수정합니다.

## 수정 내용

1. 내 할 일 데이터와 WBS 데이터를 합쳐서 표시
   - 기존에는 `myTasks`가 있으면 `wbsTasks`를 버려서, 같은 프로젝트 업무 2개만 보이는 것처럼 보일 수 있었습니다.
   - 이제 `myTasks + wbsTasks`를 id 기준으로 합쳐서 표시합니다.

2. 캘린더 표시 기준 변경
   - 기존: 계획 시작일~종료일 사이 모든 날짜에 노출
   - 수정: 마감일 기준 날짜에 표시
   - 예: 7/7 마감 업무는 7/7 칸에 표시됩니다.

3. 캘린더 우측에 `일정 등록/변경 가능 업무` 영역 추가
   - 일정 없는 업무: 일정 등록
   - 일정 있는 업무: 일정 변경
   - 버튼 클릭 시 기존 일정관리 모달 오픈

4. 캘린더 상단에 `+ 일정 등록` 버튼 추가
