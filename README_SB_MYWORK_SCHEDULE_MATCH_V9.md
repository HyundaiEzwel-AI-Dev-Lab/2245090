# PMS SB MyWork Schedule Match v9

SB 캡처 기준으로 내업무/캘린더/일정관리 팝업을 다시 맞춘 패치입니다.

## 반영 내용

- 캘린더 우측 영역을 SB 기준 `일정 미등록 업무`로 복원
- 일정 미등록 업무에는 `일정 등록` 버튼 표시
- 일정 등록된 업무에는 `일정 관리`로 변경 가능
- 일정관리 팝업을 SB 구조에 맞게 재구성
  - WBS 정보 표
  - 일정관리 안내
  - 계획일정 최초등록
  - 계획일정 등록 후 실행일정 착수/완료
  - 일정변경 요청
  - 변경사유/결재선/승인요청
  - 추가정보: 단위테스트 사용, 확정여부, 우선순위, 난이도, 비고
- 일정변경 요청 확인 팝업 추가
